黑苹果系统安装教程（一）

发表于 2020-05-03 更新于 2020-07-29
本文字数： 1.5k 阅读时长 ≈ 1 分钟

写在前面：
目前笔记本想体验黑苹果，必须是intel处理器，AMD的CPU安装太费劲而且经常出现未知的bug。并且笔记本自带的显卡大多无法驱动，笔记本的N卡连webdriver都用不了，所以只能使用核显卡。而且大多数笔记本的无线网卡无法驱动，可以通过USB网卡替代或者直接更换网卡。建议更换的网卡型号为：DW1830、DW1560、DW1820（部分可用）、BCM94360CS2(拆机网卡)等，具体更换教程可以根据不同笔记本来定，有的板载网卡的只能用USB替代（USB的可使用小米随身wifi、360随身WiFi2代、COMFAST的）这次的教程我分两期来写，大家不要着急。

首先判断机子是否符合安装

英特尔酷睿CPU(笔记本赛扬，奔腾无法驱动显卡除非只有独显)。

支持UEFI启动。

显卡：笔记本须为酷睿CPU才可以驱动核显 想要驱动独显前提为BIOS能屏蔽核显

本文只讨论inter处理器的笔记本电脑和台式机，一些高端笔记本可以驱动双显卡，特殊机型不做讨论，AMD处理器电脑也不做讨论。

镜像的选择

本教程为原版安装，去黑果小兵官网下载此镜像教程以安装macOS High Sierra 10.13.6(17G65)为例。（现在版本已经出到了10.15.4。）
本镜像含有引导，对于小白可以直接使用里面对应的config试试。如果不行其他问题请百度解决。
准备工作

优盘8G以上。

检查磁盘是否符合安装原版，硬盘分区格式。

下载镜像。

DMG写入工具：Etcher。

1.选择镜像。

2.插u盘，插上会自动识别。

3.点Flash开始写镜像，会提示给予管理员权限，给予就好。时间根据u盘读写速度而定。（写入镜像会格式化u盘，请提前做好备份）

4.正在写入，耐心等待提示完成。


将要安装macOS的磁盘分区表转换为GUID，再建立esp分区，建议300m起步。
u盘启动

将u盘插在电脑的usb2.0接口上设置为u盘启动

1.bios需要关闭安全启动（secure boot）。

2.需要bios打开uefi。

3.部分台式机bios下需要将启动系统改为其他。

4.打开AHCI，一般电脑默认打开了的。

BIOS情况复杂，只写了必要的几个设置。

安装

由于本镜像集成了黑苹果的引导，所以大家写入镜像后就可以直接u盘启动安装试试，运气好是可以直接进入安装界面的，如果运气不好卡住了另作讨论说明。

重启优盘UEFI启动 记得一定是UEFI优盘启动然后会看到如下的界面。

选择安装盘 如图红色框（看名字）按空格.

勾选Verbose(-v),此项也就是啰嗦模式和Don`t reboot on panic(debug=0x100)此项能够让安装遇到问题不会重启方便查看问题。

根据啰嗦模式的错误进行修改引导文件。

进入安装界面后先打开磁盘工具进行抹盘.

选择要安装Mac的分区 抹掉。

名字用英文，格式选择apfs，方案选择GUID。（我这里是全盘抹掉的，要抹掉单个分区选择单个分区点抹掉即可，如果在此提示错误 说分区小那是ESP分区小于200m）
此处还要拔网线，改时间，自行百度。

进行安装

这里就不说了 也就是同意 继续的 如果是英文界面 有两个选项的都选右边那个

选择刚抹掉的分区安装

选择后开始安装，这里会很快重启，重启后继续u盘启动。


重启后会多出一个或多个启动项，选择Boot macOS Install from “你的磁盘名字”会车安装，安装很快还会再次重启，然后再选择这个启动项正常安装。当再次重启后应该就安装完成了，
会有一个Boot macOS from “你的磁盘名字”的启动项，悬着他启动就可以正常进入黑苹果开机引导界面了。

后面则为正常设置，至此黑苹果的安装就完成了。
