抓包教程

发表于 2020-05-02 更新于 2020-07-29

抓包旧版请准备以下软件

1 iTunes一键卸载工具

2 Fiddler2 下载地址：http://www.telerik.com/fiddler

3 iTunes旧版本 12.3.2.35版本

4 苹果旧版app下载 （要文件的进q群：704998637）

教程开始

1.首先打开 iTunes一键卸载工具 卸载电脑上的新版iTunes（如果装了旧版就没这个必要了，你懂的）卸载完打开 我的电脑 点 音乐 把iTunes这个文件删除！

2.安装证书 打开Fiddler2 打开软件会有一个对话框提示，然后点NO，如果没有就忽略这一步！

3.安装证书开始 ，首先打开 Fiddler ，然后点击Fiddler首页顶部的 Tools 选择 Fiddler Web Debugger，进去之后勾选 Capture HTTPS CONNECTs 和 Decrypt HTTPS traffic 以及 Check for certificate revocation 三项！点击OK，提示安装证书，一直点确定或者好或者是即可！


然后点击 Actions 点 Trust root…，点 YES。再次点 Actions 点 Export root…点 确定 然后点OK 可以关闭这个软件了！

4.安装证书 这下看桌面上多了一个证书 FiddlerRoot.cer 打开，安装证书，点 将所有的证书放入下列存储，选择浏览，选择 受信任的根证书颁发机构，确定安装。

5.下载软件开始，打开旧版的iTunes并登陆你的苹果账号密码，搜索你想下载的软件，比如QQ，先搜索出来放着，然后打开软件 苹果旧版app下载 搜索QQ点击QQ软件右键 查看历史版本，就会看见右边好多版本，选择你想下载的版本号右键 下载此版本 点 确定，这下打开 旧版的iTunes，可以点QQ下面的下载了，下载出来就是旧版，点 我的应用 就可以看见刚才下载的QQ了，拉到桌面上就可以看见版本号了，然后用各大助手安装比如PP，爱思,iTools，随你吧！

6.想下载什么就先打开iTunes先搜索出来，然后用 苹果旧版app下载这个软件 搜索并点下载旧版本，具体参考5

故障处理
1.下载不了显示网络错误，修改电脑的DNS为114.114.114.114，（自己百度吧，不详细写修改DNS）然后打开C:\Windows\System32\drivers\etc 删除hosts这个文件！打开IE浏览器，点Internet选项，连接，局域网设置，去掉 为LAN使用代理服务器 前面的勾！确定就可以了！

2.只要是卸载完 安装或打开出现 Library.itl 这个文件报错的问题 那是因为你没有删除 音乐里面的iTunes文件。

3.关闭所有杀毒软件和安全卫士再使用

4.如果下载出来的还是新版软件，可能是证书没装好，或者就是自己电脑系统问题需要从新做系统.

                                                 如果你还是学不会，那就别捣鼓了！
