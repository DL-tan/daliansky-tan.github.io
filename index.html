从clover转入open core

发表于 2020-07-29 更新于 2021-04-04
本文字数： 4.7k 阅读时长 ≈ 4 分钟

众多黑苹果驱动的作者已经停止对Clover的兼容支持，改向Opencore的兼容。所以也许在未来的某一天，Clover就越来越难用了。提前了解了解OpenCore，还是很有必要的。但是OpenCore毕竟没有Clover成熟,同样也没有很好用的可视化编辑器（OpenCore Configurator会导致config出错，慎用），fix的选项也没有，对hotpatch的要求就高了。经过很多天的摸索，总结一些OC的坑，帮助以后的朋友少走一些弯路。

我的配置（这里借用朋友的笔记本，也正好锻炼一下自己的实力！！）
雷 神 G170P 笔 记 本，蓝 天(CLEVO)P170SMA模 具
CPU：Core i7-4710MQ 2.50GHz
主 板：CLEVO P17SM-A
硬 盘：威刚SSD 256GB
内 存：16 GB ( 威 刚 DDR3L 1600MHz )
显 卡：Nvidia GTX970M（DSDT屏蔽） Intel HD4600（此独显无法在高于10.13.6的系统下驱动，所以屏蔽掉）
有 线 网 卡：瑞 昱 RTL8168/8111/8112
无 线 网 卡：博通BCM94352HMB（原 装intel网 卡，无 解）
声 卡：Realtek ALC892

黑苹果系统安装教程（一）

发表于 2020-05-03 更新于 2020-07-29
本文字数： 1.5k 阅读时长 ≈ 1 分钟

写在前面：
目前笔记本想体验黑苹果，必须是intel处理器，AMD的CPU安装太费劲而且经常出现未知的bug。并且笔记本自带的显卡大多无法驱动，笔记本的N卡连webdriver都用不了，所以只能使用核显卡。而且大多数笔记本的无线网卡无法驱动，可以通过USB网卡替代或者直接更换网卡。建议更换的网卡型号为：DW1830、DW1560、DW1820（部分可用）、BCM94360CS2(拆机网卡)等，具体更换教程可以根据不同笔记本来定，有的板载网卡的只能用USB替代（USB的可使用小米随身wifi、360随身WiFi2代、COMFAST的）这次的教程我分两期来写，大家不要着急。
